PHASE 2

GradeService with an Event

Module 8
1 Add GradeService at the bottom of the file
GradeService — fires an event when a grade is posted
class GradeService { // Event — fires when a grade is posted // Action carries: student name (string) and marks (int) public event Action<string, int> OnGradePosted; public void PostGrade(TrackedStudent student, int marks) { student.Marks = marks; OnGradePosted?.Invoke(student.Name, marks); } }
2 Subscribe handlers and post grades in Run()
Wire up and post grades — inside Run()
// Create the service GradeService gradeService = new GradeService(); // Subscribe two notification handlers gradeService.OnGradePosted += EmailNotify; gradeService.OnGradePosted += LogNotify; // Post grades for each student Console.WriteLine("Posting grades..."); int[] marksData = { 88, 95, 45, 72 }; for (int i = 0; i < students.Count; i++) { gradeService.PostGrade(students[i], marksData[i]); } // ────────────────────────────────────────────── // Handler methods — write inside Capstone2 class static void EmailNotify(string name, int marks) { string grade = marks >= 90 ? "A" : marks >= 75 ? "B" : marks >= 50 ? "C" : "F"; string alert = ""; if (grade == "A") alert = " ← top scorer alert!"; if (grade == "F") alert = " ← at-risk student alert!"; Console.WriteLine("[EVENT] Email: " + name + " scored " + marks + " (Grade: " + grade + ")" + alert); } static void LogNotify(string name, int marks) { Console.WriteLine("[EVENT] Log : Grade posted — " + name + ": " + marks); }

PHASE 3

Reports via Interface

Module 9 — interfaces
1 Define the interface and two report classes
IReportGenerator + two implementations — bottom of file
// The contract — any report must implement Generate interface IReportGenerator { void Generate(List<TrackedStudent> students); } // Implementation 1 — summary only class SummaryReport : IReportGenerator { public void Generate(List<TrackedStudent> students) { Console.WriteLine("--- Summary Report ---"); Console.WriteLine("Total students : " + students.Count); double avg = students.Average(s => s.Marks); Console.WriteLine("Average marks : " + avg); var top = students.OrderByDescending(s => s.Marks).First(); var bottom = students.OrderBy(s => s.Marks).First(); Console.WriteLine("Highest mark : " + top.Marks + " (" + top.Name + ")"); Console.WriteLine("Lowest mark : " + bottom.Marks + " (" + bottom.Name + ")"); int passCount = students.Count(s => s.Marks >= 50); int passRate = passCount * 100 / students.Count; Console.WriteLine("Pass rate : " + passRate + "%"); } } // Implementation 2 — every student on one line class DetailedReport : IReportGenerator { public void Generate(List<TrackedStudent> students) { Console.WriteLine("--- Detailed Report ---"); foreach (var s in students) { Console.WriteLine(s.Id + " | " + s.Name.PadRight(8) + " | " + s.Marks + " | " + s.GetGrade() + " | " + s.Status); } } }
2 Add a method that runs any report, then call both
RunReport method + calls inside Run()
// Method accepts any IReportGenerator — works for both classes static void RunReport(IReportGenerator report, List<TrackedStudent> students) { Console.WriteLine(); report.Generate(students); } // In Run() — after posting grades RunReport(new SummaryReport(), students); RunReport(new DetailedReport(), students);

// After the reports Console.WriteLine(); Console.WriteLine("--- LINQ Analytics ---"); // Top scorers (>= 75) var topScorers = students .Where(s => s.Marks >= 75) .OrderByDescending(s => s.Marks) .ToList(); Console.Write("Top scorers (>= 75): "); foreach (var s in topScorers) Console.Write(s.Name + " (" + s.Marks + ") "); Console.WriteLine(); // At-risk students (failing) var atRisk = students.Where(s => s.Marks < 50).ToList(); Console.Write("At-risk students : "); foreach (var s in atRisk) Console.Write(s.Name + " (" + s.Marks + ") "); Console.WriteLine(); // Grade distribution using Dictionary Dictionary<string, int> gradeCounts = new Dictionary<string, int> { { "A", 0 }, { "B", 0 }, { "C", 0 }, { "F", 0 } }; foreach (var s in students) gradeCounts[s.GetGrade()]++; Console.Write("Grade distribution: "); foreach (var pair in gradeCounts) Console.Write(pair.Key + "=" + pair.Value + " "); Console.WriteLine();

CHECKLIST

Is your Capstone 2 complete?

  • StudentStatus enum is used in the TrackedStudent class
  • TrackedStudent uses properties (not public fields)
  • GetGrade() returns the correct letter for all mark ranges
  • GradeService has an event that fires when a grade is posted
  • Two handlers are subscribed: email notify and log notify
  • Both handlers run for every student when grades are posted
  • IReportGenerator interface is defined and implemented twice
  • RunReport() accepts IReportGenerator — not a specific class
  • LINQ correctly finds top scorers, at-risk students, and grade distribution
